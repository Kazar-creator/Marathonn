## Core Features
* **Multi-Category Tracking**: Separate logging for Movies, TV Shows (with episode counters), and Anime.
* **Smart Watchlists**: Users can categorize titles into *Watched*, *Watching*, or *Plan to Watch*.
* **Ratings & Reviews**: Stars or numerical rating system with user text reviews and comments section.
* **Analytics Dashboard**: Dynamic profile statistics showing total watch time, completed titles, and average ratings breakdown.
* **User Profiles**: Secure authentication and custom preferences.
